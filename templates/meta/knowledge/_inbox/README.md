# Inbox

Dateien hier ablegen → Claude verarbeitet sie bei der nächsten Session.

**Was hierher kommt:**
- PDFs, Screenshots, Exporte aus anderen Tools
- Rohe Notizen, Copy-Paste aus Dokumenten
- Design-Dateien, Styleguides
- Tabellarische Daten (CSV, Excel)

**Was Claude damit macht:**
1. Kernwissen extrahieren
2. Strukturiertes Markdown in den richtigen Ordner schreiben (`brand/`, `marketing/`, `customers/`)
3. Original-Datei nach `assets/` verschieben
